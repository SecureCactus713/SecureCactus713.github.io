Network Security Objective 3

Construct, implement and document a script or a program to automate

a security-related process or other tasks such as installation,

administration, management, mapping resources, logon scripts,

patch management, updates, auditing, analysis and assessment.

Objective Application

My NTS 370 Shell Scripting for Hackers class was the
perfect place to learn about and demonstrate this
particular objective. During this class I learned about
creating shell scripts and basic programming concepts
specifically to aid in security related tasks. My final
assignment in this class was the best complete example
of the objective.

For the Final I was tasked with creating a script to accomplish a useful
action while also meeting a number of requirements. I opted to create
a menu based script that would parse out pieces of info from a password
file for analysis, would search through a log file for failed login
attemps, and would connect to a remote web server to do port scanning.
With my my presented code I showed that I had programmed working script
for this objective.
