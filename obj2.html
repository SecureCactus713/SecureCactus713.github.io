Network Security Objective 2





Install, configure and test security hardware and software

tools with supporting documentation such as port scanners,

vulnerability detection systems, intrusion detection

systems, firewalls, system hardening, anti-virus tools,

patch management, auditing and assessment.





Objective Application





I was able to meet this objective over the course
of my NTS330 Applied Exploits and Hacking class.
I completed three assignments that operated under
a continuing scenario as a contracted Penetration
tester as well as my Final Assignment for the class.



The scenario for three of the assignments was that
I was a hired as a contract Penetration tester.
In each of the assignments I was tasked with proving
my PEN testing knowledge by completing a task.










The first assignment had me setting up a pentest platform and documenting
my process every step of the way. I was also tasked with answering several
questions to prove that my test platform was functional.

The second assignment was to gather information on an organization and
to document my findings. I was to not do any direct scanning but use
other progams as a means of data gathering on my chosen target.

The third assignment was to perform network sniffing of a closed network
and document my process and findings. I was also tasked with performing
packet capture analysis of a selected malware and documenting those findings.











The final assignment was to take the penetration
testing skills that I had learned during this course
and to apply them in a practical and legal manner.










In this assignment I was to successfully join and gain entry to the website
Hack The Box. The only way to gain entry to this site was with procuring an
invite code by attempting to hack the webpage.

This required analyzing the source code of the landing page for the site
and following the clues that were left. Some of these clues required
inspecting code with a debugger, decryipting serval types of encryptions,
and applying critical thinking and problem solving skills learned in this class.
