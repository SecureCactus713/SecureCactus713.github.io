Network Security Objective 1





Create a network infrastructure design communications

document that includes identified hardware components,

connections to outside world, identified physical

layer connectivity (media) and addressing, including

operational and security components in the design.









Objective Application





My best example of applying this objective is with the Final Assignment
from my class NTW216 Foundations of System Administration.



The final assignment was to generate a proposal for a security program
implementation plan that had to meet various parameters for a fictitious copmany.









In this assignment I created a network design that listed out hardware,
showed where said hardware would connect in the topology of the network,
and covered time and cost for implementation. The assignment was presented
in a word document in the format of a project write up as well as a Powerpoint
slide show to showcase the highlights of my recommendations.


I feel this project covers the objective with the inclusion of a network map
and appropriate visuals.
