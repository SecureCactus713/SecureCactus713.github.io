KEYS





















This project sits within the Network Security technical field. The goal of this keylogger project is to help educate people
on the importance of security practices by displaying how a simple malware can be a dangerous thing.
The way this project is making a keylogger program innovative is by having a viewable window opened when the program is running.
This viewable window will show what is being captured and how it is logging it.
